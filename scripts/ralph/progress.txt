- Files changed: cli/src/commands/claude.ts (rewritten to add spawn + polling)
- Spawns `claude <args>` with `stdio: 'inherit'` for full interactivity
- Polls `~/.claude/projects/` every 2s using recursive search for newest .jsonl file created after spawn
- Uses `birthtimeMs` to identify files created after process start
- Exports `findTranscriptFile()` and `getDetectedTranscriptPath()` for sync watcher (US-009/010)
- Forwards SIGINT/SIGTERM to child, exits with child's exit code
- One final transcript detection attempt on child exit
- **Learnings for future iterations:**
  - `spawn("claude", args, { stdio: "inherit" })` keeps Node event loop alive — no need for async main
  - `birthtimeMs` is reliable on macOS for detecting newly created files; works on modern Linux too
  - `detectedTranscriptPath` is module-level state — US-009 sync watcher should access via `getDetectedTranscriptPath()`
  - Signal forwarding: forward SIGINT/SIGTERM to child, and on child exit re-raise signal if terminated by signal
